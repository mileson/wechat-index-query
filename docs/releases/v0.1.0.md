# v0.1.0

## Highlights

- Initial public packaging for the `wechat-index-query` macOS skill bundle
- Added bilingual repository documentation and contribution guidance
- Added first release workflow, issue templates, and baseline open-source community files
- Added repository screenshots for the WeChat Index result page and OpenClaw summary output

## Installation

This repository is currently distributed as a source skill bundle. Clone or download it, then run the scripts directly on macOS with WeChat desktop and Xcode Command Line Tools installed.

## Verification

```bash
python3 -m py_compile scripts/*.py
python3 scripts/run_wechat_index_report.py --help
python3 scripts/probe_wechat_index.py
```

## How To Use

Single keyword:

```bash
python3 scripts/run_wechat_index_report.py "车牌查询"
```

Compare mode:

```bash
python3 scripts/run_wechat_index_report.py --compare "查车牌" "车牌查询" "挪车电话"
```

Screenshot mode:

```bash
python3 scripts/run_wechat_index_report.py --image /absolute/path/to/wechat-index.png
```
